# Fundamentos da computação quântica - Algoritmo de Grover

Implementação do Algoritmo de Grover usando apenas NumPy, sem dependência de frameworks quânticos.
O projeto elabora as portas quânticas do circuito para dois qubits a partir de princípios como vetores de estado, produto matricial e produto tensorial.


## Instalação

```bash
pip install numpy matplotlib
```
